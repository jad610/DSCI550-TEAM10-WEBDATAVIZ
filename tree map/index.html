Tree Map of Topic Occurrences and Category
The tree map visualized the overall count of topic occurrences in the Pixstory dataset to supplement HW1-Q1
The text at the top of the tree map shows the respective topic, the number inside parenthesis indicates the number of occurrences for the topic, and the color of the topic indicates the broader category we have placed it in.
The legend on the size indicates how the topics correspond to colors.
